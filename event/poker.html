Rules to be followed

  1. Register using the above link
  2. Once registered, a free ticket will be auto assigned to the user ID
  3. On the day of the tournament, you need to login with the same user id and password
  4. After logging in, you need to go to My Account and then to Account Summary and to Tourney Tickets
  5. You need to claim the ticket from there
  6. After that You need to click on Play Poker, the game lobby will open up now
  7. Then you need to click on Tourney and search the tournament in the name of NIT Goa (will be live from 15th Jan onwards)
  8. Next You need to click on Join and click on Register now, click on the check box
  9. Now you are all set to play the tournament.

      10. Offline

      The contest will be in Texas Hold 'Em Poker.
      Starting stack will be 4000 per Player.
      The initial blinds will start from 50/100 and the blind interval will be in 10mins.
